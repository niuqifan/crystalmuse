# Aura Nova Gems — Zodiac Crystals SEO Page

This package includes a complete SEO landing page for **Zodiac Crystals / Astrology Stones**, matching the Aura Nova Gems visual style: cream background, deep plum headings, gold accents, rounded luxury cards, starry crystal imagery, dry flowers, candles, and purple silk.

## Files

- `index.html` — browser preview page
- `assets/zodiac-crystals.css` — page styling
- `assets/zodiac-crystals.js` — search and element filter behavior
- `shopify-section-zodiac-crystals.liquid` — Shopify custom section version
- `assets/images/` — hero, overview, and 12 zodiac card images
- `image-assets-preview.jpg` — all image assets preview

## Recommended SEO

URL:
`/pages/zodiac-crystals` or `/pages/zodiac-crystal-guide`

SEO title:
`Zodiac Crystals Guide: Best Crystals for Each Zodiac Sign | Aura Nova Gems`

Meta description:
`Find the best crystals for your zodiac sign. Explore zodiac crystal meanings, astrology stones, birth sign energy, and crystal recommendations for Aries to Pisces.`

H1:
`Zodiac Crystals: Find the Perfect Stone for Your Sign`

## Shopify Usage

1. Upload all files inside `assets/images/` into your Shopify theme assets.
2. Create a new section named `zodiac-crystals.liquid`.
3. Paste the content from `shopify-section-zodiac-crystals.liquid`.
4. Add this section to a new page template.
5. Set the page URL to `/pages/zodiac-crystals`.

## Collection URL Notes

The page uses collection URLs such as:

- `/collections/aries-crystals`
- `/collections/taurus-crystals`
- `/collections/fire-sign-crystals`
- `/collections/water-sign-crystals`

Update these handles if your Shopify collection URLs are different.
