# Aura Nova Gems — Chakra Guide SEO Page

This package includes a complete SEO landing page for a Chakra Guide page matching the Aura Nova Gems visual style: warm cream background, luxury serif headings, plum + gold accents, rounded cards, and soft chakra crystal imagery.

## Files

- `index.html` — browser preview page
- `assets/chakra-guide.css` — page styling
- `assets/chakra-guide.js` — search and chakra filter behavior
- `shopify-section-chakra-guide.liquid` — Shopify custom section version
- `assets/images/` — hero, overview, and 7 chakra card images

## Recommended SEO

URL:
`/pages/chakra-guide` or `/chakra-guide/`

SEO title:
`Chakra Guide: 7 Chakras, Meanings & Healing Crystals | Aura Nova Gems`

Meta description:
`Explore the 7 chakras, their meanings, colors, energy signs, and the best healing crystals for balance, protection, love, clarity, and spiritual alignment.`

H1:
`Chakra Guide: Balance Your Energy With Healing Crystals`

## Shopify Usage

1. Upload all files inside `assets/images/` into your Shopify theme assets.
2. Create a new section named `chakra-guide.liquid`.
3. Paste the content from `shopify-section-chakra-guide.liquid`.
4. Add this section to a new page template.
5. Set the page URL to `/pages/chakra-guide`.

## Notes

The shop links use collection URLs such as `/collections/root-chakra`, `/collections/heart-chakra`, and `/collections/7-chakra-crystals`. Update them if your Shopify collection handles are different.
