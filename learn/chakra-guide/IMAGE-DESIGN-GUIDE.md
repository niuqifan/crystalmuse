# Chakra Guide 图片部分设计说明

## 设计目标
这版图片部分专门匹配 Aura Nova Gems 当前的页面风格：奶油白背景、薰衣草紫丝绸、暖色虚化光点、干花装饰、原石/水晶柱质感，以及深紫 + 金色的高级感。

## 已设计图片资源

### 页面核心图片
- `chakra-guide-hero.webp` — 首屏 Hero 横幅，7 个脉轮水晶按红、橙、黄、绿、蓝、靛、白紫排列。
- `chakra-overview-crystals.webp` — “What Are Chakras?” 区块配图，展示多种水晶合集。
- `chakra-guide-og-image.webp` — SEO / 社交分享图，适合 Open Graph 和 Twitter Card。
- `chakra-guide-card-thumbnail.webp` — 可用于 Learning Hub 或入口卡片的小图。

### 7 个脉轮卡片图片
- `card-root-chakra.webp` — 红色/黑色系，适配 Root Chakra。
- `card-sacral-chakra.webp` — 橙色系，适配 Sacral Chakra。
- `card-solar-plexus-chakra.webp` — 金黄色系，适配 Solar Plexus Chakra。
- `card-heart-chakra.webp` — 绿粉色系，适配 Heart Chakra。
- `card-throat-chakra.webp` — 蓝色系，适配 Throat Chakra。
- `card-third-eye-chakra.webp` — 靛紫色系，适配 Third Eye Chakra。
- `card-crown-chakra.webp` — 白紫透明系，适配 Crown Chakra。

## 图片规格
- 页面横幅和卡片图：`1600 × 760 px`
- 社交分享图：`1200 × 630 px`
- Learning Hub 卡片缩略图：`900 × 520 px`
- 格式：`WEBP`
- 风格：柔光、丝绸、干花、暖色 bokeh、原石质感

## 使用建议
1. Shopify 中请把 `assets/images/` 里的所有图片上传到主题 Assets。
2. 页面 Hero 使用 `chakra-guide-hero.webp`。
3. Learning Hub 的 “Chakra Guide” 卡片可以使用 `chakra-guide-card-thumbnail.webp`。
4. SEO 分享图可以使用 `chakra-guide-og-image.webp`。
5. 如果之后做 Crystal Meanings / Zodiac / Intention 页面，建议继续沿用同一套背景和光影风格，保持品牌统一。
