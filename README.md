# AURANOVAGEMS Healing Energy Catalog

A deployable static landing page and product catalog for a crystal healing / OEM wholesale brand.

## Files

- `index.html` — main page
- `styles.css` — visual design and responsive layout
- `script.js` — filters, product modal, inquiry basket and demo forms

## GitHub Pages Deployment

1. Create a new GitHub repository.
2. Upload all files in this folder to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Choose branch `main` and folder `/root`.
6. Save, then open the GitHub Pages URL after deployment finishes.

## Notes

- Forms are demo-only. Connect to Formspree, EmailJS, Shopify, WooCommerce, or your backend before collecting real inquiries.
- Product data is currently in `script.js`, so it is easy to edit without a database.
- All visuals are CSS-generated gradients, so the site works without external image files.
